# CH 3 숙련 Spring_일정 관리 앱 Develop
### 최길중(Spring_3기)

-----
## 📌 프로젝트 소개
Spring Boot + JPA로 구현한  일정 관리 REST API 입니다.  
일정은 생성/조회/수정/삭제(CRUD)가 가능하며, 도전 기능으로 댓글 기능과 입력값 검증 까지 구현했습니다.  
수정/삭제는 비밀번호 검증을 통해 보호되며, 작성일/수정일은 JPA Auditing 으로 자동 관리됩니다.

-----

## 📑 Features
- 일정 CRUD (생성/전체조회/단건조회/수정/삭제)
- 작성자명 조건 조회, 수정일 기준 내림차순 정렬
- 비밀번호 검증으로 수정/삭제 보호
- 댓글 생성, 일정 단건 조회 시 댓글 목록 포함
- 입력값 검증 (제목 30자, 내용 200자, 댓글 100자, 필수값)
- JPA Auditing으로 작성일/수정일 자동 관리
- API 응답에서 비밀번호 제외
